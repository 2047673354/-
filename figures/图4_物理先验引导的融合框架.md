# 图4 物理先验引导的融合框架（放在"三、物理先验与深度学习的融合框架"中部）

```mermaid
flowchart LR
subgraph 输入
X1[相位/幅度序列(多子通道)]
X2[角-距-时间张量]
X3[微多普勒时频图]
Q[质量特征: SNR/PLV/谐波一致性]
end

subgraph 编码
E1[序列编码器(1D CNN+BiGRU)]
E2[张量编码器(3D CNN)]
E3[时频编码器(2D CNN)]
end

X1 --> E1
X2 --> E2
X3 --> E3
Q --> A[门控/注意力(质量引导)]
E1 --> A
E2 --> A
E3 --> A
A --> T[Transformer 融合]
T --> Y1[呼吸率 RR]
T --> Y2[心率 HR]
T --> Y3[置信度/体动段]
```

图注：多模态输入经专用编码器后，由质量引导的注意力先筛选再融合，并输出RR/HR与置信度。
